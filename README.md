# AI Image Caption Generator

This project is an AI-powered Image Caption Generator developed using Vision Transformer (ViT) and GPT-2 models from Hugging Face Transformers.

## Features
- Automatic image caption generation
- Deep learning based architecture
- Real-time image upload and caption prediction
- Google Colab support

## Technologies Used
- Python
- PyTorch
- Transformers
- PIL
- Matplotlib

## Installation

pip install transformers torch torchvision pillow matplotlib

## Model Used
nlpconnect/vit-gpt2-image-captioning
